불연속 함수

  • 함수의 값이 중간에 갑자기 변하는 함수
  • 부호함수 : 입력이 양수이면 1, 음수이면 -1, 0이면 0
  • 단위계단함수 : x=0에서 불연속인 함수
  • 지시함수 :
    • 함수 이름 아래 첨자로 미리 지정된 값이 들어오면 출력이 1이 되고 아니면 출력이 0이 된다.

함수의 그래프

  • 함수의 형상을 시각적으로 파악하기 위해 그래프 또는 플롯을 사용하기도 한다.
  • 구간간격이 너무 크면 그래프가 부정확해지고, 구간 간격이 너무 작으면 계산 시간이 증가하여 메모리 등의 리소스가 낭비된다.

역함수의 그래프

  • 역함수의 그래프는 원래의 함수에서 x축과 y축이 바뀐 것이므로 y=x 가 나타내는 직선을 대칭축으로 대칭인 함수의 그래프와 같다.

데이터 분석에서 많이 사용되는 함수들

  • 다항식 함수 : 상수항, 일차항, 이차항 등의 거듭제곱 항의 선형조합으로 이루어진 함수
  • 최대함수와 최소함수
  • 지수함수 : 밑을 오일러로 하여 거듭제곱을 하는 함수
    - 양수를 거듭제곱한 값이므로 항상 양수다.
- x=0 일때 1이 된다.
- x가 양의 무한대로 가면, 양의 무한대로 다가간다.
- x가 음의 무한대로 가면, 0으로 다가간다.
- x1>x2 이면 expx1 > expx2이다.
  • 로지스틱 함수 : 지수 함수를 변형한 함수로 회귀분석이나 인공신경망에서 자주 사용된다.
  • 로그 함수 : e를 거듭제곱하여 특정한 수 a가 되도록 하는 함수
    • x값, 즉, 입력변수 값이 양수이어야 한다. 0이거나 음수이면 정의되지 않는다.
    • x>1 이면 y>0
    • x=1 이면 y=0
    • 0<x<1 이면 y<0
    • x1>x2면 logx1>logx2
    • 로그 함수의 성질
      • 로그함수는 곱하기를 더하기로 변환한다.
      • 어떤 함수에 로그를 적용해도 함수의 최고점, 최저점의 위치는 변하지 않는다.
      • 로그함수는 0부터 1사이의 작은 값을 확대시켜 보여준다.
  • 소프트플러스 함수:
    • 지수함수와 로그함수를 결합하여 만든 함수로 0을 인수로 갖는 최대함수와 비슷하지만 x=0 근처에서 값이 부드럽게 변한다는 장점
  • 다변수 함수
    • 복수의 입력변수를 가지는 함수
    • 2차원 함수는 평면상의 지형과 같기 때문에 3차원의 서비스 플롯, 또는 컨투어 플롯으로 나타낼 수 있다.
  • 분리 가능 다변수함수:
    • 단변수함수의 곱으로 표현 가능한 다변수 함수
  • 다변수 다출력 함수
    • 입력변수 뿐만 아니라 출력변수도 여러개인 함수
    • 출력 벡터의 특성
      • 모든 출력원소는 0과 1사잇값을 갖는다.
      • 모든 출력원소의 합은 1이다.
      • 입력 원소의 크기 순서와 출력 원소의 크기 순서가 같다.

Module

Module

  • 파이썬에서 모듈은 def를 사용하여 정의한다.
  • def가 실행되면, 함수의 객체와 참조가 같이 생성된다.
  • 반환값을 정의하지 않으면, 파이썬은 자동으로 None을 반환한다.

1. 스택과 활성화 레코드

  • 함수가 호출될 때마다 활성화 레코드가 생성된다.
  • 활성화 레코드는 다음과 같은 순서로 처리된다.
    1. 함수의 실제 매개변수를 스텍에 저장한다.
    2. 변환 주소를 스택에 저장한다.
    3. 스택의 최상위 인덱스를 함수의 지역변수에 필요한 총량만큼 늘린다.
    4. 함수로 건너뛴다.
  • 활성화 레코드를 풀어내는 절차는 다음과 같다.
    1. 스택의 최상위 인덱스는 함수에 소비된 총 메모리양만큼 감소한다.
    2. 반환 주소를 스택에서 빼낸다.
    3. 스택의 최상위 인덱스는 함수의 실제 매개변수만큼 감소한다.

2. 모듈의 기본값

  • 모듈을 생성할 때, 함수 또는 메서드에서 가변 객체를 기본값으로 사용해서는 안된다.
  • 따라서 다음과 같이 사용하는 것이 좋다.
[7]

3. __ init __ .py 파일

  • 패키지는 모듈과 __ init__.py 파일이 있는 디렉터리이다.
  • from 폴더이름 import * : 이름이 __ 로 시작하는 모듈을 제외한 모듈의 모든 객체를 불러온다.
  • python -c import 모듈 : 터미널에서 특정 모듈이 있는지 확인할 수 있다.

4. name 변수

  • 파이썬은 모듈을 임포트 할 때마다 __ name __이라는 변수를 만들고, 모듈 이름을 저장한다.

5. 컴파일된 바이트코드 모듈

  • 컴파일러가 사용하는 바이트 컴파일 코드는 표준모듈을 많이 사용하는 프로그램의 시작시간을 줄이기 위한 것이다.

딕셔너리 매서드

setdefault

  • setdefault() : 딕셔너리에서 키의 존재 여부를 모른 채 접근할 때 사용된다. (딕셔너리에 존재하지 않는 키에 접근하면 예외가 발생한다.)
  • A.setdefault(key, default) 를 사용하면 딕셔너리 A에 key가 존재할 경우 키에 해당하는 값을 얻을 수 있고, key가 존재하지 않는다면, 새 키와 기본값 default가 딕셔너리에 저장된다.
{'key1': ['value1', 'value2'], 'key2': ['value1', 'value2']}

update

  • A.update(B)는 딕셔너리 A에 딕셔너리 B의 키가 존재한다면, 기존 A의 (키,값)을 B의 (키,값)으로 갱신한다. B의 키가 A에 존재하지 않는다면, B의 (키,값)을 A에 추가한다.
{'a': 1, 'b': 10}

get

  • A.get(key)는 딕셔너리 A의 key값을 반환한다. key가 존재하지 않으면 아무것도 반환하지 않는다.

PCA

PCA

  • 데이터 간 변이 규칙을 찾아낼 때 PCA를 이용한다.
  • 주성분 분석 또는 차원축소라고도 한다.
  • 가능한 한 쓸모 있는 정보를 유지하면서 더 적은 차원의 찬원축소 벡터로 선형변환 하고자 하는 것이다.
  • PCA의 목표는 변환결과인 차원축소 벡터가 원래의 벡터가 가졌던 정보와 가장 유사하게 되는 변환행렬 값을 찾는 것이다.

사이킷런의 PCA 기능

  • 사이킷런의 decomposition 서브패키지는 PCA분석을 위한 PCA 클래스를 제공한다.
    • 입력 인수 :
      • n_components : 정수
    • 매서드 :
      • fit_transform() : 특징행렬을 낮은 차원의 근사행렬로 변환
      • inverse_transform() : 변환된 근사행렬을 원래의 차원으로 복귀
    • 속성 :
      • mean_ : 평균벡터
      • components_ : 주성분 벡터

Eigen_decomposition

고윳값 분해

  • 고윳값과 고유벡터를 찾는 작업
  • N차원 정방행렬 A에 대해
    1. 행렬 A는 N개의 고윳값-고유벡터를 가진다.
    2. 행렬의 대각합은 모든 고윳값의 곱과 같다.
    3. 행렬의 행렬식은 모든 고윳값의 곱과 같다.
    4. 행렬 A가 대칭행렬이면 N개의 실수 고윳값을 가지며 고유벡터들이 서로 직교한다.
    5. 행렬 A가 대칭행렬이고 고윳값이 모두 양수이면 양의 정부호이고 역행렬이 존재한다. 역도 성립한다.
    6. 행렬 A가 어떤 행렬 X의 분산행렬 X.T X이면 0 또는 양의 고윳값을 가진다.
    7. 행렬 X가 풀랭크이면 분산행렬 X.T X는 역행렬이 존재한다.

대각화

  • 만약 고유백터행렬의 역행렬이 존재한다면 행렬을 고유벡터행렬과 고윳값행렬의 곱으로 표현할 수 있다.
  • 행렬을 대각화할 수 있으면 대각화가능 행렬이라고 한다.
  • 행렬이 대각화가능하려면 고유벡터가 선형독립이어야 한다.
  • 대각화가능한 행렬에 0인 고유값이 없으면 항상 역행렬이 존재한다.
  • 행렬 A가 실수인 대칭행렬이면 고유값이 실수이고 고유벡터는 서로 직교한다.
  • 실수인 대칭행렬은 항상 대각화 가능하다.

대칭행렬의 고유분해

  • 행렬 A가 실수인 대칭행렬이면 고유값이 실수이고 고유벡터는 서로 직교한다.
  • 실수인 대칭행렬은 항상 대각화가능하다.
  • 대칭행렬이 양의 정부호이면 고윳값은 모두 양수이다. 역도 성립한다.
  • 대칭행렬이 양의 준정부호이면 고윳값은 모두 0이거나 양수다. 역도 성립한다.

분산행렬

  • 행렬 X에 대해서 X.T X인 정방행렬을 분산행렬이라고 한다.
  • 분산행렬은 양의 준정부호이고 고유값은 0보다 같거나 크다.
  • 행렬 X가 풀랭크이면 이 행렬의 분산행렬 X.T X의 역행렬이 존재한다.
