딕셔너리 매서드

setdefault

  • setdefault() : 딕셔너리에서 키의 존재 여부를 모른 채 접근할 때 사용된다. (딕셔너리에 존재하지 않는 키에 접근하면 예외가 발생한다.)
  • A.setdefault(key, default) 를 사용하면 딕셔너리 A에 key가 존재할 경우 키에 해당하는 값을 얻을 수 있고, key가 존재하지 않는다면, 새 키와 기본값 default가 딕셔너리에 저장된다.
{'key1': ['value1', 'value2'], 'key2': ['value1', 'value2']}

update

  • A.update(B)는 딕셔너리 A에 딕셔너리 B의 키가 존재한다면, 기존 A의 (키,값)을 B의 (키,값)으로 갱신한다. B의 키가 A에 존재하지 않는다면, B의 (키,값)을 A에 추가한다.
{'a': 1, 'b': 10}

get

  • A.get(key)는 딕셔너리 A의 key값을 반환한다. key가 존재하지 않으면 아무것도 반환하지 않는다.

items, values, keys

  • items(), keys(), values() 메서드는 딕셔너리 뷰다.
  • 딕셔너리 뷰란 딕셔너리의 항목을 조회하는 읽기 전용의 반복 가능한 객체다.
dict_items([('name', 'jhon'), ('age', 17), ('hobby', 'game')])
dict_values(['jhon', 17, 'game'])
dict_keys(['name', 'age', 'hobby'])
{'name': 'jhon', 'age': 17, 'hobby': 'game', 'address': 'seoul'}

pop(), popitem()

  • A.pop(key)는 딕셔너리 A의 key 항목을 제거한 후, 그 값을 반환한다.
  • A.popitem()은 딕셔너리 A에서 항목(키와 값)을 제거한 후, 그 키와 항목을 반환한다.

clear

  • 딕셔너리의 모든 항목을 제거한다.

딕셔너리 순회

  • 반복문에서 딕셔너리를 순회할 때는 기본적으로 키를 사용한다. 딕셔너리의 키는 임의의 순서대로 나타나지만, sorted() 함수를 사용하면 정렬된 상태로 순회할 수 있다.
  • sorted()함수는 딕셔너리의 keys(), values(), items()에 대해 사용할 수 있다.

Set

set

  • 반복 가능하고, 가변적이며, 중복요소가 없고, 정렬되지 않은 컬렉션 데이터 타입이다.
  • 셋은 멤버십 테스트 및 중복 항목 제거에 사용된다.

set method

  • add() : 셋 A에 x가 없는 경우 x를 추가한다.
  • update() : A.update(B)는 A를 B에 추가한다.
  • union() : update 메서드와 같지만, 연산결과를 복사본으로 반환한다.
  • intersection()과 &연산자 : 교집합의 복사본을 반환한다.
  • difference()와 -연산자 : 차집합의 복사본을 반환한다.
  • clear() : 모든 항목을 제거한다.

PCA

PCA

  • 데이터 간 변이 규칙을 찾아낼 때 PCA를 이용한다.
  • 주성분 분석 또는 차원축소라고도 한다.
  • 가능한 한 쓸모 있는 정보를 유지하면서 더 적은 차원의 찬원축소 벡터로 선형변환 하고자 하는 것이다.
  • PCA의 목표는 변환결과인 차원축소 벡터가 원래의 벡터가 가졌던 정보와 가장 유사하게 되는 변환행렬 값을 찾는 것이다.

사이킷런의 PCA 기능

  • 사이킷런의 decomposition 서브패키지는 PCA분석을 위한 PCA 클래스를 제공한다.
    • 입력 인수 :
      • n_components : 정수
    • 매서드 :
      • fit_transform() : 특징행렬을 낮은 차원의 근사행렬로 변환
      • inverse_transform() : 변환된 근사행렬을 원래의 차원으로 복귀
    • 속성 :
      • mean_ : 평균벡터
      • components_ : 주성분 벡터

Eigen_decomposition

고윳값 분해

  • 고윳값과 고유벡터를 찾는 작업
  • N차원 정방행렬 A에 대해
    1. 행렬 A는 N개의 고윳값-고유벡터를 가진다.
    2. 행렬의 대각합은 모든 고윳값의 곱과 같다.
    3. 행렬의 행렬식은 모든 고윳값의 곱과 같다.
    4. 행렬 A가 대칭행렬이면 N개의 실수 고윳값을 가지며 고유벡터들이 서로 직교한다.
    5. 행렬 A가 대칭행렬이고 고윳값이 모두 양수이면 양의 정부호이고 역행렬이 존재한다. 역도 성립한다.
    6. 행렬 A가 어떤 행렬 X의 분산행렬 X.T X이면 0 또는 양의 고윳값을 가진다.
    7. 행렬 X가 풀랭크이면 분산행렬 X.T X는 역행렬이 존재한다.

대각화

  • 만약 고유백터행렬의 역행렬이 존재한다면 행렬을 고유벡터행렬과 고윳값행렬의 곱으로 표현할 수 있다.
  • 행렬을 대각화할 수 있으면 대각화가능 행렬이라고 한다.
  • 행렬이 대각화가능하려면 고유벡터가 선형독립이어야 한다.
  • 대각화가능한 행렬에 0인 고유값이 없으면 항상 역행렬이 존재한다.
  • 행렬 A가 실수인 대칭행렬이면 고유값이 실수이고 고유벡터는 서로 직교한다.
  • 실수인 대칭행렬은 항상 대각화 가능하다.

대칭행렬의 고유분해

  • 행렬 A가 실수인 대칭행렬이면 고유값이 실수이고 고유벡터는 서로 직교한다.
  • 실수인 대칭행렬은 항상 대각화가능하다.
  • 대칭행렬이 양의 정부호이면 고윳값은 모두 양수이다. 역도 성립한다.
  • 대칭행렬이 양의 준정부호이면 고윳값은 모두 0이거나 양수다. 역도 성립한다.

분산행렬

  • 행렬 X에 대해서 X.T X인 정방행렬을 분산행렬이라고 한다.
  • 분산행렬은 양의 준정부호이고 고유값은 0보다 같거나 크다.
  • 행렬 X가 풀랭크이면 이 행렬의 분산행렬 X.T X의 역행렬이 존재한다.

  • 정방행렬이 아닌 행렬은 고유분해가 불가능하지만 특이분해를 할 수 있다,
  • 특이 분해는 모든 행렬에 대해 가능하다.
  • 특잇값의 개수는 행렬의 열과 행의 개수 중 작은 값과 같다.
